# Fumin 🌙→🟠

![Fumin — 不眠・眠らないMac](docs/images/hero.png)

**不眠 — 眠らないMac。**

MacBookのフタを閉じても、中の作業を止めずに動かし続けるための小さな無料アプリです。
メニューバー（画面右上の時計のあたり）に住んでいて、**クリックひとつ**でON/OFFを切り替えられます。

## これは何のためのアプリ？

MacBookは、フタを閉じると自動的に「スリープ」（おやすみモード）に入ります。
ふだんは便利な仕組みですが、こんなとき困ります:

- 🤖 **AIに長い作業を任せている**（Claude Code や Codex にコードを書かせている途中）
- 📥 大きなファイルを**ダウンロード中**
- 🎬 動画の書き出しや**バックアップの最中**

フタを閉じたとたんに全部止まってしまうからです。

Fuminは、この「フタを閉じたら寝ちゃう」をOFFにできるスイッチです。
机の上でフタを閉じたまま、Macに仕事を続けさせられます（カバンに入れるのはダメ。[下の注意](#使うときの注意大事)を読んでね）。

## アイコンの意味

- 🌙 `moon.zzz`（通常色）= **通常モード**（フタを閉じると普通にスリープ）
- 🟠 オレンジのカップ = **スリープ無効モード ON**（フタを閉じても起き続ける）

「コーヒーを飲んで起きてる☕」と覚えてください。

## インストール（ダウンロードするだけ）

1. [**Releasesページ**](https://github.com/Yanagi-1112/fumin/releases/latest)を開き、`Fumin.zip` をダウンロード
2. ダウンロードしたzipをダブルクリックで解凍 → 出てきた `Fumin.app` を「**アプリケーション**」フォルダにドラッグ
3. `Fumin.app` をダブルクリックで起動

> ⚠️ 初回起動時に「開発元を確認できないため開けません」と出たら:
> **システム設定 → プライバシーとセキュリティ** を開き、下の方に出ている「"Fumin"は…」の右にある「**このまま開く**」を押してください。
> これはApple税（年間$99の開発者登録）を払っていない個人アプリすべてに出る警告で、中身はこのページで全部公開されているコードそのものです。

## 初回セットアップ（1回だけ・1行コピペ）

「フタを閉じても寝ない」設定はmacOSの深い場所にあるため、Fuminに変更の許可を与える必要があります。
初めてアイコンをクリックすると案内が出るので、「コマンドをコピー」を押して、**ターミナル**（Launchpadで「ターミナル」と検索）に貼り付けてEnterを押してください。Macのログインパスワードを聞かれます（入力しても画面には表示されませんが、ちゃんと入っています）。

```sh
echo "$(whoami) ALL=(root) NOPASSWD: /usr/bin/pmset" | sudo tee /etc/sudoers.d/fumin >/dev/null && sudo chmod 440 /etc/sudoers.d/fumin
```

これは「**pmsetというAppleの省電力設定コマンドだけ**、パスワードなしで実行してよい」という許可を書き込むものです。それ以外の権限は一切与えません。

## 使い方

- **左クリック** … ON/OFFをトグル（🌙⇄🟠）
- **右クリック**（またはControl+クリック）… メニュー

メニューでできること:

| 項目 | 説明 |
|---|---|
| 画面をすぐ消す | 画面だけオフにする（中の作業は動き続ける） |
| フタを閉じたら画面も消す | 外部モニタ運用（クラムシェル）で、フタを閉じたらモニタも自動で暗くする |
| ログイン時に自動起動 | Macを起動したらFuminも自動で立ち上げる |

### よくある使い方: AIに作業させたまま離席する

1. Fuminをクリックして 🟠 にする
2. フタを閉じる
3. Macは起きたまま、作業が続く。戻ってきたらフタを開けて、Fuminをクリックして 🌙 に戻す

## 安全性について

「よくわからないアプリに設定をいじらせるの、怖くない？」——正しい感覚です。だからFuminは全部オープンにしています:

- 📖 **ソースコード全公開**: このアプリの中身は [`Fumin.swift`](Fumin.swift) の1ファイルだけ。約200行で、プログラマーなら数分で全部読めます
- 🌐 **インターネットに一切つながない**: 通信するコードがそもそも存在しません。何かを送信したり、情報を集めたりしません
- 🔓 **触るのはスリープ設定だけ**: Appleの正規コマンド `pmset` を呼ぶだけで、ファイルやデータには触れません
- 🛡 **三重の安全装置**: OFFにしたとき・アプリを終了したとき・起動したとき、必ず「普通にスリープする状態」に戻します。「気づいたら永久にスリープしないMacになってた」事故を防ぎます
- 💰 **無料・広告なし**（MITライセンス）

## 困ったときの魔法の呪文

何かおかしいと感じたら、ターミナルでこれを実行すれば即・通常状態に戻ります:

```sh
sudo pmset -a disablesleep 0
```

## 使うときの注意（大事）

- 🔌 フタ閉じ＋長時間は**必ず電源アダプタに接続**する
- 🔥 **カバン・引き出し・布団の上は厳禁**。フタを閉じていてもMacは動いていて熱を出します。硬くて風通しのよい机やスタンドの上で使ってください

## アンインストール

```sh
osascript -e 'quit app "Fumin"' 2>/dev/null
rm -rf /Applications/Fumin.app
sudo rm -f /etc/sudoers.d/fumin
```

## 仕組み（技術メモ）

- ON: `sudo -n pmset -a disablesleep 1`（フタ閉じスリープを無効化）＋ `caffeinate -imsu`（アイドルスリープ抑止）
- OFF / 終了 / 起動時: 必ず `pmset -a disablesleep 0` に戻す
- 画面消し: `pmset displaysleepnow`（root不要）。フタの開閉は `IOPMrootDomain` の `AppleClamshellState` で検出。キーボード・マウスに触れると画面は復帰する

- macOS 13 (Ventura) 以降 / Apple Silicon・Intel 両対応
